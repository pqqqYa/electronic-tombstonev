# 神经网络的架构

input → Weights → Summation and Bias → Activation → Output

* input输入
* Weights权重
* Summation and Bias每个数据与不同权重相乘后加和偏差
* Activation激活函数
* Output

当损失函数收敛到最小后得到最终结果
