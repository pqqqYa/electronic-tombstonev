# 最大公因数

~~~c
for (int i=1;i<=n;i++)
    {
        if(m%i==0&&n%i==0)
        {
            x = i;
        }
    }
~~~

从0开始加，上限为较小值，没有求出来则最大共因素为1

# 最小公倍数

~~~c
for (int i=n;;i++)
    {
        if(i%m==0&&i%n==0)
        {
            z = i;
            break;
        }
    }
~~~

从最小共因素开始加，无上限，求导值再退出
