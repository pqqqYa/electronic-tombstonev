### 4.1.2 if-else语句

* if(表达式)为真(1)，就执行这个语句；反之则不执行。
* if，else if，else实现多分支语句。
* 括号写法注意规范，方便阅读。
~~~c
if(Expression)
{
	Functions
}else{
	Functions
}
~~~
