### 7.1.1 函数的声明与定义

函数间的调用关系是，由主函数（main函数）调用其他函数，其他函数也可以互相调用，但是其他函数不可以调用main函数。同一个函数可以被一个或多个函数调用任意次。

* 声明函数
* 调用函数（在main函数或者其他函数中）
* 定义函数

~~~c
#include <stdio.h>  
void print_message();//声明函数  
int main()  //主函数
{  
    print_message();//调用函数  
    return 0;  
}  
void print_message()//定义函数  
{  
    printf("How do you do\n");  
}
~~~
