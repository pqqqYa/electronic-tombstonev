### 基本信息

请求路径：/user
请求方式：GET
接口描述：返回用户数据

### 请求参数

请求参数格式：queryString
请求参数说明：

| 参数名称 | 说明   | 类型     | 是否必须 | 备注  |
| ---- | ---- | ------ | ---- | --- |
| id   | 主键ID | number | 是    |     |
请求数据样例

~~~json
id =6
~~~

### 响应数据

响应数据类型：application/json
响应参数说明：

| 名称   | 类型     | 是否必须 | 默认值 | 备注            | 其他信息 |
| ---- | ------ | ---- | --- | ------------- | ---- |
| code | munber | 必须   |     | 响应码：0-成功，1-失败 |      |
| msg  | string | 必须   |     | 提示信息          |      |
| data | object | 非必须  |     | 返回的数据         |      |
